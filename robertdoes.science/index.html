I use bioinformatics and machine learning to figure out ageing and other tricky questions.


> Why study ageing?
Because it is both challenging and absolutely fascinating.
Ageing is a catch-all for all the things that will break in the human body given enough time, often connected to age-related diseases such as cancer, diabetes, and Alzheimer's disease. Each a complex problem in its own right, they are also interconnected in complicated ways. This makes them a perfect fit for computational approaches designed to wrangle meaning from complex and noisy data.

> What have you worked on?
I have analysed and interpreted high throughput biological data to investigate the tissue-specific insulin response in mutant flies, highlighting specific metabolic responses that make them live longer. More recently, I have worked with RNAseq data from mouse embryonic stem cell (mESC) lines, teasing apart regulatory networks of self-renewal (paper in preparation).

> Web apps using R Shiny
This server was set up to tinker with the shiny web framework for R. As part of the ongoing project investigating self-renewal of mESC's, I have build an app that allows interactive exploration of data and results. In the meantime, I will host some smaller apps here:

[CellPlot] generates segmented bar charts. These visualize functional term enrichments (e.g. from the gene ontology analysis of a gene set), and simultaneously the relative changes of individual genes between experimental conditions.

[shinySETGO] determines if a geneset of interest is enriched in known biological functions (gene ontology terms). Redundancy is reduced by clustering terms that contain the same subsets of genes. I wrote the underlying script to simplify these standard analyses in my own projects. Implementing the web app served as a refresher in mySQL.
